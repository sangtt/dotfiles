# Dotfiles
My dotfiles
I made it work on ubuntu

# How to install

Run the following command:

```sh
$ bash -c "$(curl -fsSL https://raw.github.com/sangtt/dotfiles/master/install)"
```

## How to update

Run the install command:

```sh
$ bash install
```
